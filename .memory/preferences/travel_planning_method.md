# 旅行攻略思考方法

## 核心公式
- 游览模式（效率优先）：幸福总量/总时间 = (目标地点幸福 + 交通幸福) / 总时间
- 度假模式（幸福总量优先）：幸福总量 = 幸福效率 × 总时间

## 规划顺序
### 游览模式
1. 跨市交通（往返方式）
2. 市内交通
3. 确定天数
4. 确定景点（候选→确定）
5. 定酒店
6. 找餐厅/商圈
7. 排每日行程

### 度假模式
1. 交通
2. 酒店
3. 候选景点和餐厅
4. 行李清单
5. 完成（可选：排详细行程）

## 关键洞察
- 目标地点的单位时间幸福收益最高，交通通常是负幸福
- 度假模式核心：先最小化不快乐，再延长时间
- 景点分"候选"和"确定"两阶段，需根据时间约束筛选
