# OSINT 调查认知修正

## 零推测规则

永远不要仅根据用户名相似性推断身份。必须满足以下条件之一才能确认身份：
- 存在经过验证的加密或跨平台引用
- 用户明确确认
- 无法找到时明确标注「未找到经过验证的账户」

## 跨语言搜索规则

当调查双语实体时：
- 中文查询不足以覆盖全部平台
- 必须将中文查询翻译成英文进行二次搜索
- 特别是对于 Medium、GitHub 等国际平台，优先使用英文搜索
