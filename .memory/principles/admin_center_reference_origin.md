# 景点距离参考原点

## 用途

为行政单位选定一个参考原点，用于衡量该单位内各景点相对于"旅行者最可能落脚的位置"的距离。距离越远，旅行者前往该景点的意愿成本越高。

## 方法

取该行政单位的**政府所在地**（市政府、县政府、区政府）作为参考原点。

通过高德地图 POI 搜索 API，关键词 `{行政单位名}政府`，参数 `citylimit=true`，取排名第一的结果的经纬度。

不加"政府"二字直接搜地名也可以（高德对行政单位的 POI 坐标就是政府所在地），但部分小地方直接搜地名可能无结果（如额济纳旗），加"政府"更可靠。

## 已知局限：行政中心搬迁

约44%的省会城市（12/27）市政府已迁至新区，偏差从5km到25km不等。搬迁后酒店、美食、历史景点仍集中在老城区。暂不特殊处理：省会只占行政单位总数极小比例，地级市和县级搬迁比例远低于省会，且偏差方向一致不影响相对比较。

- 本方法仅用于距离的相对比较，不是精确旅行时间计算依据。
